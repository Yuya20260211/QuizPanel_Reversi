# QuizPanel Reversi

QuizPanel Reversi は、クイズ大会・配信企画・身内イベントで使える、アタック25風のクイズ用オセロ盤アプリです。

出題者は問題文と答えを見ながら進行し、回答者には番号とジャンルだけを見せられます。正解者が出たらその人の色でマスを取り、オセロのように挟んだマスをひっくり返します。

## どんなアプリ？

クイズ番組っぽい盤面を、PC上で手軽に動かすためのデスクトップアプリです。

- 出題者側パネル: 問題文、答え、正解者選択、スコア、保存などを操作
- 回答者側パネル: 参加者に見せる盤面専用ウィンドウ
- CSV読み込み: 自作の問題リストをそのままゲームに使用
- オセロ判定: 正解者の色で挟んだマスを自動で反転
- 自動保存: 途中で落ちても再開しやすい

会場のプロジェクターや配信画面には回答者側パネルを映し、手元のPCでは出題者側パネルを操作する想定です。

## 主な機能

- 出題者側と回答者側の2ウィンドウ表示
- 任意サイズの盤面
- 初期設定画面の盤面初期値は5 x 5
- CSVから問題を読み込み
- 問題番号をCSVの行順から自動生成
- シャッフルなし、シャッフルあり
- 正解者あり、正解者なしの両方に対応
- オセロ判定によるマス反転
- 取得マス数によるスコア表示
- プレイヤー名入力欄は選択した色に連動
- 回答者側のスコア数値表示ON/OFF
- 回答者側のジャンル表示ON/OFF
- クイズ開始からの経過時間表示
- 予備問題は未使用のものをCSV順に使用
- 答えをクリックで表示、または常時表示
- 出題済問題一覧
- Undo
- 灰色戻し
- 手動保存
- 1ターンごとの自動保存
- JSONから再開

## 画面の役割

### 出題者側パネル

進行担当が操作する画面です。

- 灰色マスをクリックして問題を表示
- 問題文と答えを確認
- 正解者を選択
- 正解者なしを選択
- Undo
- 灰色戻し
- スコア確認
- 回答者側パネルの表示復帰
- 出題済問題一覧の確認
- 手動保存
- ゲーム終了/結果発表

答えは、初期状態では隠しておけます。答え欄をクリックすると表示されます。`常に答えを表示` をONにすると、答えを最初から表示できます。

### 回答者側パネル

参加者や視聴者に見せる画面です。

- 盤面を表示
- 問題番号を表示
- ジャンルを表示
- 必要に応じてスコアを表示

問題文と答えは表示されません。

## 基本の流れ

1. アプリを起動します。
2. CSVファイルを選びます。
3. 行数と列数を指定します。
4. プレイヤー人数、名前、色を設定します。
5. シャッフル方法を選びます。
6. ゲームを開始します。
7. 出題者側で灰色マスをクリックします。
8. 問題を読み上げます。
9. 正解者が出たら、その人のボタンを押します。
10. 正解者なしの場合は `正解者なし` を押します。
11. Turnが1進み、問題が出題済みになります。

問題を表示しただけではTurnは進みません。Turnが進むのは、正解者あり・正解者なしを確定したときだけです。

## CSV形式

CSVの基本形式は次の3列です。

```csv
ジャンル,問題文,答え
スポーツ,野球でバッターがノーヒットで出塁する四球を英語で何と言う？,フォアボール
歴史,鎌倉幕府を開いた人物は？,源頼朝
アニメ,スタジオジブリ作品で、森の主である巨大な生き物と少女たちの交流を描いた作品は？,となりのトトロ
```

### 列について

| 列 | 内容 |
| --- | --- |
| ジャンル | 回答者側にも表示される分類 |
| 問題文 | 出題者側だけに表示される問題 |
| 答え | 出題者側だけに表示される答え |

問題番号列は不要です。問題番号はCSVの行順から自動で付きます。

### ヘッダーについて

次のようなヘッダーに対応しています。

- `ジャンル,問題文,答え`
- `カテゴリ,問題,解答`
- `category,question,answer`

ヘッダーなしの3列CSVも読み込めます。

### 空欄の扱い

- ジャンルが空欄: `ノージャンル` として扱います。
- 問題文が空欄: エラーになります。
- 答えが空欄: エラーになります。

### 文字コード

以下のCSVに対応しています。

- UTF-8
- UTF-8 BOM付き
- CP932 / Shift-JIS 系

Excelで作ったCSVでも読み込みやすいようにしています。

## 盤面サイズと必要問題数

盤面サイズはゲーム開始時に指定します。

| 盤面 | 必要な初期問題数 |
| --- | ---: |
| 5 x 5 | 25問 |
| 6 x 6 | 36問 |
| 8 x 8 | 64問 |

必要問題数は `行数 x 列数` です。

CSVの先頭から `行数 x 列数` 問が初期盤面に使われます。それ以降の問題は予備問題です。

## 予備問題

正解者なしなどで灰色のまま残ったマスを再クリックした場合、予備問題が出題されます。

予備問題は、ジャンルに関係なく未使用のものをCSV順に使います。

例えば、最初のマスがアニメでも、予備問題の先頭がスポーツならスポーツが出ます。
このとき、出題者側と回答者側のパネル表示は、左上のマス番号は固定のまま、ジャンルだけ予備問題のジャンルに切り替わります。

## シャッフル

ゲーム開始時にシャッフル方法を選べます。

| 設定 | 内容 |
| --- | --- |
| シャッフルなし | CSVの順番どおりに使います |
| シャッフルあり | CSV全体をシャッフルして使います |

シャッフルした場合、元CSVは上書きしません。シャッフル後のCSVが別名で保存され、そのCSVをゲームに使います。

## 回答者側の表示設定

出題者側パネルから、回答者側の見せ方を切り替えられます。

### 回答者側にスコア数値を表示する

回答者側パネルには、回答者名と色が常に表示されます。

ONにすると、回答者側パネルにも取得枚数を表示します。

OFFの場合、回答者名と色だけを表示し、取得枚数は隠します。途中経過の枚数だけ隠したいときに使えます。

### ジャンルを隠す

ONにすると、回答者側パネルではジャンルを隠し、番号だけを表示します。

出題者側パネルではジャンルを表示したままです。出題者は情報を持ち、回答者には情報を絞る。

## 出題済問題一覧

`出題済問題一覧` ボタンから、すでに確定した問題を確認できます。

表示される内容:

- 番号
- ジャンル
- 問題
- 答え

正解者あり、正解者なしのどちらでも、確定した問題は一覧に入ります。予備問題も対象です。

この一覧情報はJSON保存にも含まれます。自動保存の `game_backup.json` と手動保存JSONのどちらにも保存されます。

## Undo

`戻る` または `Ctrl + Z` で、操作を戻せます。

戻る対象:

- 盤面の色
- Turn
- 使用済み問題
- 予備問題の使用状況
- 正解者なしの状態
- 灰色戻し
- スコア
- 出題中の問題
- 回答者側スコア数値表示設定
- 回答者側ジャンル非表示設定

間違えて押しても戻せるようにしています。

## 灰色戻し

ボーナス問題などで、色付きマスを1つ灰色に戻せます。

- Turnは進みません。
- オセロ判定は発生しません。
- 周囲のマスの色は変わりません。
- そのマスの問題は再利用しません。
- 戻したマスを再クリックすると予備問題が出ます。
- Undo対象です。

## 保存と再開

### 自動保存

1ターン進むごとに、自動でJSON保存します。

自動保存ファイル:

```text
game_backup.json
```

保存場所は、読み込んだCSVと同じフォルダです。

### 手動保存

出題者側パネルの保存ボタンから、任意の場所にJSON保存できます。

### 再開

起動画面からJSONファイルを読み込むことで、保存時点から再開できます。

保存される主な内容:

- 盤面サイズ
- 使用CSVパス
- 元CSVパス
- 問題シャッフル設定
- プレイヤー情報
- 盤面の色
- 各マスの初期番号と初期ジャンル
- 使用済み問題
- 出題済問題一覧
- Turn
- 操作履歴
- スコア数値表示設定
- 回答者側ジャンル非表示設定
- 現在出題中の問題

## 終了条件

結果発表は、出題者が `ゲーム終了/結果発表` を押したときに表示されます。

全マスが取得されても、自動では結果発表へ進みません。

予備問題が必要な場面で使える問題がなくなった場合は、「問題が無くなりました」と表示します。この通知だけでは結果発表へは進みません。

## よくある困りごと

### CSVが読み込めない

以下を確認してください。

- 問題文が空欄になっていないか
- 答えが空欄になっていないか
- 盤面サイズに対して問題数が足りているか
- CSVが3列になっているか

### 文字化けする

Excelから保存したCSVの場合、CP932 / Shift-JIS 系として読み込めるようにしています。それでも崩れる場合は、CSVをUTF-8で保存し直してください。

### 回答者側パネルを閉じてしまった

出題者側パネルの `回答者パネルを表示` から再表示できます。

### 答えが最初から見えてしまう

`常に答えを表示` がONになっていないか確認してください。

### 予備問題のジャンルが違う

予備問題はジャンルに関係なくCSV順に使います。予備問題が出題されたマスは、左上のマス番号は固定のまま、ジャンルだけ予備問題の内容に変わります。

